## Raw Data

This project uses high-resolution lunar imagery from:

- **LROC NAC (Lunar Reconnaissance Orbiter Camera)**
- **Chandrayaan-2 TMC-2**

Due to large file sizes and data licensing constraints, raw satellite datasets
are not included in this repository.

### Data Sources
- LROC NAC: https://wms.lroc.asu.edu
- Chandrayaan-2 TMC-2: https://www.issdc.gov.in

Researchers can download the datasets directly from the official portals.
